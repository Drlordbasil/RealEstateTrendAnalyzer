To run the Python program you shared, you would need the following package requirements:
- requests
- beautifulsoup4
- pandas
- matplotlib
- seaborn

You can install these packages using pip by running the following command:

```
pip install requests beautifulsoup4 pandas matplotlib seaborn
```

Please note that these command assumes you have pip already installed on your system. If not, you can refer to the pip documentation to install it.
